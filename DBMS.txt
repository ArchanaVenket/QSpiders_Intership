	DATBASE:	
		Database is a software which will manage the Database.	
		The 2 important features provided by the DBMS are
			1.Security
			2.Authorization

		Types Of DMBS:
			1.Hierarchical Database Management System
			2.Network Database Management System
			3.Object Oriented Database Management System
